OpenCart ↔ ERP
Products

Inventory

Push current stock quantities from ERP → OpenCart.

Orders

Amazon ↔ ERP
Products

Full product sync from ERP → Amazon (all items with sync enabled).

Inventory

Push current stock quantities from ERP → Amazon via Feeds API.

Orders

Import new orders from Amazon SP-API → ERP Sales Orders.

Category Mappings
Re-sync to OpenCart

Re-push all items in mapped item groups to OpenCart with updated category assignments.

Re-sync to Amazon

Re-push all items in mapped item groups to Amazon with updated product type assignments.

Bench Commands (CLI)

Run these inside the ERP-app container:

# OpenCart
bench --site erp.local execute erpmin_integrations.opencart.product.full_product_sync
bench --site erp.local execute erpmin_integrations.opencart.inventory.sync_all_inventory
bench --site erp.local execute erpmin_integrations.opencart.order.import_orders

# Amazon
bench --site erp.local execute erpmin_integrations.amazon.product.full_product_sync
bench --site erp.local execute erpmin_integrations.amazon.inventory.sync_all_inventory
bench --site erp.local execute erpmin_integrations.amazon.order.import_orders
